Focify Icon Generator

This tool generates PNG icons from the SVG for your Chrome extension.

Instructions:
  1. Click "Generate Icons" to create all sizes
  2. Right-click each canvas and "Save image as..."
  3. Save as icon16.png, icon32.png, icon48.png, and icon128.png
  4. Replace the files in the assets/icons folder
